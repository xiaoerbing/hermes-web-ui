# 课题需求 × Hermes Web UI 可开发实现功能清单

> 课题：《国家中小学智慧教育平台支撑下的教育数字资源共享与应用研究》

## 概述

本文档将课题 6 大核心模块中的每一功能需求，与 Hermes Web UI **现有能力** 和 **可扩展能力** 进行精确映射，产出可直接进入开发的实现方案。

## 模块三：教师教学辅助 & 资源二次开发 — 可开发功能

### 3.1 资源解构五步法工具 ★核心

这是课题最核心的创新点，Hermes 可以完美承载：

```
教师上传/选择平台资源
    ↓
AI Agent 拆解（析要素） → 关联课标（链课标） → 结合学情修改（融情境）
    ↓
一键生成改编课件（改设计） → 使用后收集反馈（评效果）
```

| 步骤 | Hermes 实现方案 |
|------|----------------|
| **析要素** | 新建 Skill `resource-deconstructor`，上传资源 → LLM 拆解知识点列表、教学要素 |
| **链课标** | 知识库存储课标文档，检索相关条目，生成"此资源对应课标X.Y.Z"映射表 |
| **融情境** | ChatView 中对话式交互，"我教的是农村初中二年级，请根据学情调整" |
| **改设计** | LLM 生成改编后的课件大纲、活动设计、分层任务 |
| **评效果** | 使用后记录评价 → 存入 SQLite → 汇聚成课例库 |

**涉及文件：**

| 文件 | 操作 | 说明 |
|------|------|------|
| `packages/skills/resource-deconstructor/SKILL.md` | CREATE | 资源解构 Skill |
| `packages/skills/curriculum-mapper/SKILL.md` | CREATE | 课标映射 Skill |
| `packages/skills/lesson-planner/SKILL.md` | CREATE | 教学设计生成 Skill |
| `packages/server/src/services/hermes/resource-deconstructor.ts` | CREATE | 五步法服务端逻辑 |
| `packages/client/src/components/hermes/education/ResourceDeconstructionPanel.vue` | CREATE | 五步法 UI 面板 |

## 总结

Hermes Web UI 已有超过 **60% 的基础能力**可以直接复用于教育课题场景：

- **聊天引擎** → AI 伴学、教师辅助、虚拟助教
- **文件管理** → 资源库基础
- **群聊** → 教研共同体
- **Kanban** → 课题进度管理
- **Cron Job** → 资源同步、报告生成
- **用量分析** → 评价数据可视化
- **技能系统** → 教育专属能力（课标解读、资源解构）
- **知识库（规划中）** → RAG 检索、课标映射

**核心新增工作量集中在：** 资源管理系统、问卷工具、学习轨迹数据库、教育专属 Skills 编写。
